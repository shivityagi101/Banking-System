🚀 Features

- Create new bank accounts
- Deposit money
- Withdraw money
- Check balance
- View transaction history 
- Error handling and input validation
- Persistent data storage using MySQL


 🛠️ Technologies Used

- **Python 3.x**
- **MySQL**
- `mysql.connector` (Python library for connecting MySQL)
